# R304_Projet_Qualite_Dev
### Voici notre projet de simulation de l'envahissement de l'Armorique.

Les fichiers sont oragnisés de la manière suivante : 
- Le répertoire `diagrams` contient les diagrammes de classes
- Le répertoire `src` comprend plusieurs packages :
    1. `asterix.characters` qui contient les classes liées aux personnages.
    2. `asterix.werewolf` avec les classes spécifiques à la gestion des lycanthropes.
    3. `asterix.food` comprend les classes pour la nourriture et la potion magique.
    4. `asterix.places` contient les différents lieux.
    5. `asterix.main` est le package dédié au théâtre d'envahissement, la classe pour lancer la simulation.

Ce projet a été réalisé par 3 étudiants dans un cadre pédagogique.
